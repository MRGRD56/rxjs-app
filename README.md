# rxjs-app
Simple RxJS examples

### Launch
Before you start the app, install Parcel CLI:  
**npm** `npm install -g parcel-bundler`  
**Yarn** `yarn global add parcel-bundler`

Then, restore packages and start the app:

#### npm
`npm install`  
`npm run start`

#### Yarn
`yarn install`  
`yarn run start`
