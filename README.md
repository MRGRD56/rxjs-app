# rxjs-app
Simple RxJS examples

### Launch
To run this app you must have Parcel CLI installed:  
**npm** `npm install -g parcel-bundler`  
**Yarn** `yarn global add parcel-bundler`

Restore packages and start the app:

#### npm
`npm install`  
`npm run start`

#### Yarn
`yarn install`  
`yarn run start`
